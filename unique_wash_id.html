Solution overview

Unique WASH ID

A Digital Public Infrastructure for improved WASH services in Liberia — giving every public WASH asset a single, persistent identifier that all agencies can reference, without replacing their existing databases.

The problem
Multiple agencies hold fragmented, incomparable data on the same assets
→
The solution
Unique WASH ID registry
→
The outcome
Harmonised, complete data for informed investments and infrastructure decisions
💡
What is it

Every public WASH asset in Liberia — natural or engineered — receives a unique, persistent ID. All agencies reference the same ID when sharing or publishing data about that asset.

Each ID is anchored to a minimal set of core parameters (GPS location, asset type). Additional detail stays in each agency's own database, which is never replaced.

The result is a digital thread connecting multiple sovereign databases — making data comparable across institutions without creating a new centralised system.

⚙
Core features
  • Asset registration and unique WASH ID generation
  • AI-powered deduplication to prevent redundant entries
  • Secure, API-based data querying across agency databases
  • Governance and user management portal
